## Getting Started

### Prerrequisitos

- Node.js 24.6.0 o mayor

### Instalacion

Instalar Dependencias:
```bash
npm install
```

### Desarrollo

Iniciar la aplicacion en modo dev:
```bash
npm run dev
```

La app va a estar disponible en `http://localhost:5173`

## Tecnologias utilizadas

- React 18.3.1
- Vite 5.4.1
